Configuración Inicial

Para sincronización en tiempo real entre dispositivos, necesitas conectar tu base de datos Supabase. Sigue estos pasos:

1

Crea tu cuenta gratuita

Ve a supabase.com → New Project → elige nombre y región.

2

Obtén tus credenciales

En tu proyecto: Settings → API → copia Project URL y anon public key

3

Ejecuta el SQL de configuración

En SQL Editor de Supabase, pega y ejecuta el script que encontrarás en README_SQL.txt incluido en este paquete.

Tus credenciales se guardan localmente solo en este navegador.
El sistema usa Supabase para sincronización entre dispositivos.
